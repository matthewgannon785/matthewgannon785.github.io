Assignment Summary

Part 1: Orthogonal Vectors

  1. Created a C++ function called isOrth to check if two integer vectors are orthogonal.
  2. Compiled the C++ function isOrth from the command line.
  3. Specified the compilation commands in the PDF file.
  4. Loaded the function into R and created an R wrapper function to call the compiled C++ function.
  5. Tested the compiled function by passing different vector combinations to the wrapper function.

  2. Written an R wrapper function signC() to call the compiled C++ function.
  3. Tested the implementation by passing a specific numeric vector to signC().
