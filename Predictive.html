LASSO / Ridge / Linear / quasi-likelihood

Assignment 1

An advanced predictive analytics assignment comprising four questions covering various statistical techniques and models.

Question 1: Utilizing the Boston dataset to predict housing values using linear regression, ridge regression, and LASSO. Tasks include data splitting, model fitting, test error assessment, and comparison of prediction methods.

Question 2: Analysis of a density function to show its belonging to the exponential family, finding the variance function, and obtaining the deviance function from a random sample.

Question 3: Deriving likelihood equations and likelihood ratio statistics for gamma distribution parameters estimation.

Question 4: Proposing and analyzing gamma and inverse-Gaussian GLMs to explain leukemia survival times. Also, exploring a quasi-likelihood model with unknown dispersion parameter.

Assignment 2

Question 1: Analyzing the impact of education level and anxious attachment on unwanted pursuit behaviors in couples post-separation. Tasks include fitting a Poisson GLM, testing for significance of covariates, assessing overdispersion, proposing alternative regression models, and evaluating predictive performance.

Question 2: Investigating math achievement among students from a sample of 5 high schools in the USA. Tasks involve fitting normal regression and mixed-effects models, computing intraclass correlation coefficient, predicting random effects, and assessing model significance.

Question 3: Modeling a dataset with a true curve and noise. Tasks include generating data and plotting the true curve, fitting curves using kernel smoothing, smoothing splines, and regression splines with different degrees of freedom, and comparing the methods for estimating the true curve.
