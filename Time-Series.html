Time Series Analysis

Ljung-Box-Pierce / ACF / ARMA / MLE / Stationarity

Assignment

Here's a brief summary of the Time Series tasks in the below assignment:

Question 1:

  1. Plot the time series and its ACF to assess stationarity.
  2. Identify an autoregressive model based on the PACF plot.
  3. Estimate model parameters using Yule-Walker method and write the AR equation.
  4. Construct confidence intervals for AR coefficients and discuss lag significance.

Question 2:

  1. Plot the time series and its ACF/PACF to analyze patterns.
  2. Propose an ARMA model based on observed patterns.
  3. Estimate model parameters using maximum likelihood estimation and assess lag significance.
  4. Use selection criterion to choose among various ARMA models.

Question 3:

  1. Assess stationarity of the time series and determine necessary differences.
  2. Propose an ARMA model and estimate parameters using maximum likelihood estimation.
  3. Assess statistical significance of included lags.
  4. Evaluate model fit using diagnostic tools and comment on residuals.
